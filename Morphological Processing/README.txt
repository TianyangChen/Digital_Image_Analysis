The proof of four properties on dilation and erosion was shown in 'proof.pdf'

The four images generated by the following 2 functions all assume that 0-pixels represent the foreground.
If the user assumes non-0-pixels represent the foreground, he just needs to set parameter 'fg' to 255.

================================================================
function dilation(InputImage,OutputImage,fg,SE_width)
================================================================
 InputImage - the file name of input image
OutputImage - the file name of output image
         fg - this parameter has two values: 
              0   denotes black region is the foreground
              255 denotes white region is the foreground
   SE_width - the width (an odd number) of the structuring element
================================================================
To implement dilation, simply type the following command line in MATLAB's command window:
	dilation('morph.png','AdB.png',0,11)
================================================================


================================================================
function erosion(InputImage,OutputImage,fg,SE_width)
================================================================
 InputImage - the file name of input image
OutputImage - the file name of output image
         fg - this parameter has two values: 
              0   denotes black region is the foreground
              255 denotes white region is the foreground
   SE_width - the width (an odd number) of the structuring element
================================================================
To implement erosion, simply type the following command line in MATLAB's command window:
	erosion('morph.png','AeB.png',0,11)
================================================================
